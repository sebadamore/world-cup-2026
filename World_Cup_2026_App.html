🔗 Connect Google Calendar API

1
Create a Google Cloud Project

Go to console.cloud.google.com → New Project. Enable the Google Calendar API for your project.

2
Create OAuth2 Client ID

APIs & Services → Credentials → Create Credentials → OAuth Client ID. App type: Web application. Add http://localhost:8080 and your exact deployed origin (for example https://world-cup-2026-one.vercel.app) to Authorized JavaScript Origins.

3
Create an API Key

Credentials → Create Credentials → API Key. Optionally restrict it to the Calendar API. Copy both the Client ID and the API Key below.

4
Configure OAuth Consent Screen

Google Auth Platform → Branding. Set User Type to External if sharing with others. Add scope https://www.googleapis.com/auth/calendar under Data Access.

Credentials are used only in your browser and never sent anywhere. This app uses the official Google Calendar JS quickstart pattern with GAPI + Google Identity Services (GIS).
